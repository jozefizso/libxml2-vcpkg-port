# libxml2 port files for vcpkg

> Build definitions of **libxml2** library for VC++ packaging tool.

This repository contains port files for building **libxml2** library using [vcpkg] tool.

Supported releases:
* 2.7.8
* 2.9.3
* 2.9.4
